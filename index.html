Document Creation

With LinearOS, you can easily create and edit documents using the DocuMaker app! To create a document, simply open DocuMaker and type in "new". After you've created your document, you can type in anything to add content to your document, use the command "exitleaveTHIS" after you're done, "saveMYFILE" to save it before you exit, and "upLOADtoINTERNpotato" to upload your document to the Internet!

LinearOS Paint

Paint is an app where you can easily draw anything, but your brush is only a black circle. You will need to use the Run Commands button to exit, erase your drawing, save your file as a .ART, or upload your image to the Internet.

Internet Features: View and Create Online Content

With the listed utility apps (DocuMaker, Paint, TextBase IDE) and their matching online viewer apps (DocuMaker Internet R, The Art Internet, TextBase Web), you can easily create and view online content! Just use the "upLOADtoINTERNpotato" command (For Paint drawings, use command "upload") to upload your document, image, or website to the Internet! To view online content via DocuMaker Internet R, The Art Internet, or TextBase Web, just type the name or link of the document, image, or website you want to see!

Settings: Utilize Prompt Boxes To Customize Your Experience

The Settings Terminal uses advanced prompt-box logic to customize your desktop experience! You can control your wallpaper, Timmy and Sophie's behavior, and even music playback with the advanced prompt-box logic of the Settings Terminal! Whether you want a 2000s power user interface or complete digital chaos, the Settings Terminal is for you when using LinearOS!

Is Server Up?: Track Whether Online Features (like login) Will Work

Wanna know whether you will be able to use the online content viewers or even log in later? The Is Server Up? app is the solution for you! This app detects if the server is up without you even having to open the app, due to the marker that changes its color to red when the backend server is down! When you do open the app, you will be met with a LinearOS-styled message that tells you if the server is working, and you can press space to exit!

Simple Login/Signup: Just Add Username

WARNING: DO NOT SHARE USERNAMES WITH OTHER LINEAROS USERS. YOU LOG IN WITH THESE USERNAMES.

LinearOS uses a simple login technique. Just type in your username, and the program tries to find it on the server! If you enter a username that is not on the server, however, you will be able to create a username on the server! You can also use the "random" command to generate a random username that is almost coherent, but still digital chaos!
